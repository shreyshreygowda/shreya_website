# Shreya Nadagowda — Personal Website

A simple, animated personal site built with vanilla HTML, CSS, and JavaScript.

## Run locally

Open `index.html` in your browser, or serve with any static server:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000).

## Deploy

Works on GitHub Pages, Netlify, Vercel, or any static host — just point it at this folder.
